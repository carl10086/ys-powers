## 背景

需要为 Claude Code 构建一个基于 qmd 的记忆系统：
- 每次解决问题后，提取关键信息生成 SOP 记忆
- 下次遇到类似问题时，AI 自动检索历史记忆供参考

## 解决方案

1. 设计 QMD 记忆系统方案 (docs/superpowers/specs/2026-04-17-qmd-memory-sop-design.md)
2. 确认 MVP 方向：先实现 sop-add command 测试效果
3. 确定核心流程：从 session 历史自动提取 → 预览确认 → 保存到 ./sop/
4. 确定实现方案：预览确认模式 (方案 B)
5. 创建 sop-add 命令文件 (commands/sop-add.md)
6. 编写设计文档并提交

## 关键文件

- `commands/sop-add.md`
- `docs/superpowers/specs/2026-04-17-qmd-memory-sop-design.md`
- `docs/superpowers/specs/2026-04-17-sop-add-command-design.md`

## 关键决策

- 从 session 历史自动提取，不需要用户额外输入
- 预览确认模式：生成后显示内容，用户确认才保存
- 输出目录: ./sop/
- 命名格式: sop-YYYYMMDD-NNN.md
