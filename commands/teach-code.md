# teach-code

带你一步步理解任意代码模块的源码，通过文档 + 交互式问答的方式引导深入细节。

## 使用方式

```
/teach-code <文件/目录/模块>
```

## 流程

### Step 1: 探索上下文

读取指定路径的代码，分析：
- 文件结构与目录组织
- 核心模块及其职责
- 依赖关系与调用链

输出整体概览（中文）。

### Step 2: 交互式问答引导

根据代码复杂度，AI 提出关键问题引导深入理解。

规则：
- **一次一问**，用户回答后逐步深入
- AI 根据回答决定下一个问题的方向
- 遇到复杂调用链或分支逻辑时，提供可视化辅助（chrome-devtools）
- 根据代码复杂度**自动决定**深入程度

### Step 3: 总结

生成代码理解笔记，包含：
- 模块概览
- 核心设计思路
- 关键实现细节

保存到 `docs/superpowers/specs/YYYY-MM-DD-<模块名>-read.md`

## brainstorming skill 的角色

本命令借助 brainstorming skill 的：
- 交互式问答框架（一次一问，循序渐进）
- 用户意图澄清能力
- 文档产出流程（生成理解笔记）
